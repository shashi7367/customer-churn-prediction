# Customer Churn Prediction System

## Project Overview
AI-powered system to predict customer churn using Machine Learning with business impact analysis.

## Results
- Model Accuracy: 87%
- Algorithm: XGBoost
- Dataset: 7,043 customers
- High Risk Customers: 892
- Revenue Saved: 6.6 Lakhs

## Tech Stack
- Python
- XGBoost
- SHAP Explainability
- Streamlit
- Plotly
- Pandas and NumPy

## Features
- Real-time churn prediction
- SHAP explainability — WHY customer will churn
- Business impact analysis
- AI-powered retention recommendations
- Interactive dashboard

## How to Run
pip install streamlit xgboost shap plotly pandas
streamlit run app.py

## Business Impact
- Total Customers: 7,043
